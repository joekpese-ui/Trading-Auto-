# Trading Auto

Application de trading automatique pour les marchés financiers.

## Objectifs
* Faire du trading sur des marchés spécifiques (Forex, actions, etc.)
* Utiliser des indicateurs techniques pour prendre des décisions de trading

## Dépendances
* Python 3.8+
* `pandas` pour l'analyse de données
* `numpy` pour les calculs numériques
* `yfinance` pour l'accès aux données financières
